MORKDOWN BODY の領域を、縦サイズを最大化できるようにする。ボタンで切り替えられるようにする。

- `MARKDOWN BODY` の横に縦サイズ最大化切り替えボタンを設置する
- ホットキーでも切り替えられるようにする
  - macOSでは Cmd + h
  - その他のOSでは Ctrl + h
- 最大化が有効の場合、TASK DETAILS 内の PRIORITY, STATUS, TITLE, RELATIVE PATH, CREATED, UPDATED の領域が非表示になり、その分 MARKDOWN BODY の textarea が広がる
