# タイトルをそのままファイル名にする

- RELATIVE PATH は taskDirs から初期値が入力された状態にする。
- TITLE を入力すると自動で RELATIVE PATH の値が `${taskDir}/${title}.md` 形式になる
- RELATIVE PATH は手動で上書きすることができる
- 手動で上書き後は RELATIVE PATH は TITLE と自動で同期されなくなる
